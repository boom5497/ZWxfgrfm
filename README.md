# SpringBoot Vue作业批改系统简介

## 核心功能
- 网站首页展示
- 作业通知发布与管理
- 作业测试进行与结果查看
- 系统公告发布与查看
- 个人中心信息管理与操作
- 学生作业提交与查看批改结果
- 教师批改作业与发布通知
- 学生成绩查看与管理
- 作业分类管理
- 班级管理
- 作业题目库管理
- 作业题目管理
- 系统管理

## 适用场景
- 教育机构内部作业批改与管理
- 教学辅助系统
- 线上教学环境下的作业处理

## 技术特点
- 采用B/S架构，实现前后端分离
- MVC模式开发，易于维护和扩展
- 基于Java、SpringBoot、Vue等主流技术栈
- 开发环境包括IDEA、JDK1.8、Mysql、Maven、Navcat等
- 提供详细的文档和视频教程，方便学习和使用
